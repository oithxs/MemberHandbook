# サービス  
HxSで使えるもの
## 各種ライセンスの配布
- Windows10 Proの無料配布
    主務に聞く
- GMSというゲーム作成ソフト
    #randomで聞く

## 部所有のサーバーの貸し出し
サーバー管理担当に連絡する

## 部内機材の貸し出し
- 原状復帰可能な部室内での使用 => 許可不要
- 原状復帰可能な部室外への持ち出し => #randomで持ち出しの報告
- 原状復帰不可能な使用 => #generalで審議

## 部室の使用：印刷、充電などができます。(一応WiFiもある)

## 勉強会の開催時にN%5==0ごとにジュース一本の支援
